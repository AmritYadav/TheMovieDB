## TheMovieDB

The App is built using MVVM and Clean Architecture pattern.

    1. App (Base Module)
    2. Presentation
    2. Domain
    3. Data
    4. Remote
    
## Tech Stack.
    1. Material Design
    2. Kotlin
    3. Android Architecture Components
    3. Retrofit
    5. Glide
